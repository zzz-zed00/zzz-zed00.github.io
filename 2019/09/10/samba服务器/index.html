samba服务器的安装和使用

samba服务器可以共享不同设备之间的文件，实现资源共享

1、安装

这里用yum安装方式

yum -y install samba samba-client

2、配置

一般情况下防火墙会禁用samba服务，这里将其设置为允许

firewall–cmd –permanent –add-service=samba

设置完成要重启防火墙服务

firewall–cmd –reload

启动samba服务

systemctl start smb.service nmb.service

3、建立共享目录

4、创建访问账号

useradd -s /sbin/nologin user

smbpasswd -a user 将该user用户添加到samba服务中

【-a 添加smb 帐号；-x 删除smb 帐号；-d 禁用smb 帐号；-e 启用smb 帐号】

5、修改配置文件

进入目录/etc/samba/smb.conf

在文件末尾添加共享文件

[company] //共享名称为company
comment = company share //共享注释
path = /home/company //指定共享路径
browseable = yes //所有人可见
guest ok = no //拒绝匿名访问
writeable = yes //支持写入数据
[company]
comment = company share
path = /home/company
guest ok = no //拒绝匿名访问
valid users=user,user1,user2 //允许访问的用户列表
write list=user //允许写入的用户列表

修改完成，重启samba服务器(systemctl restart smb nmb)

6、测试配置文件正确

testparm当配置文件有错时会给出相应提示

7、测试samba服务器

在linux 客户端访问测试:smbclient -U smbuser //192.168.137.133/company
查看服务器的共享文件有哪些：smbclient -L 192.168.137.133
查看smbuser 用户可以访问的共享文件：smbclient -L 192.168.137.133 -U smbuser

8、Linux下访问Windows文件夹的方式

在Windows上设置共享目录：d:\file

在Linux下安装samba-client客户端

​ #yum -y install samba-client

安装cifs-utils软件包

​ #yum -y install cifs-utils

linux下建立挂载点

​ #mkdir /mnt/windows

挂载d:\file到/mnt/windows

​ #mount -t cifs -o username=name password=pass //192.168.137.133/file /mnt/windows

挂载成功即可进入/mnt/Windows目录查看windows的共享文件夹

在/etc/fstab文件中添加挂载信息进行永久挂载

//192.168.137.133/redhat_disk /mnt/windows cifs username=name password=123456 0 0

至此配置完成
