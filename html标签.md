# html标签

``` html
<a href='http://baidu.com'>baidu</a> 
		定义锚  点击baidu按钮会返回定义的地址
<abbr title="People  Paksda JSAlksa">PPJ</abbr>
		定义一个缩写  在鼠标移动到定义的缩写上时会显示该缩写定义的信息
<acronym title="Jjas Kajsd Sasjdka">JKS</acronym>
		定义一个首字母的缩写，基本同上
<address>标记作者信息</address>
<area>
<article>定义文章、评论、博客、帖子等</article>
<audio str="/xx/xx.ogg" controls="controls">anjd</audio> 
		str为地址  controls播放按钮
<b>定义粗体字</b>
<base定义页面中的所有默认地址或目标>
<bdo dir="rtl">定义文本方向</bdo> 
    	输出为从右往左
<big>放大标签</big>
<br>换行
<button type="button">dd</button>按钮
<input type="button" value="asa">按钮
```
